# Reference

Material de referencia / handoff de diseño.

- El handoff original de Claude Design se recibe como
  `Scumm-handoff.zip` y contiene `theme_scumm/demo.html` +
  sprites en pixel art.
- Sprite sheet adicional en pixel art 16-bit con panel de verbos
  (MIRAR · USAR · HABLAR · COGER · DAR · ABRIR · CERRAR · EMPUJAR ·
  TIRAR), inventario 4×4, iconos (gota, sol, nube, nube con lluvia,
  ola, árbol, cursor mano) y caja de diálogo beige con texto
  "HOLA, ¿QUÉ TAL?" — paleta cian/verde/amarillo sobre marco marrón
  con esquinas doradas. Esa paleta se ha incorporado a las variables
  CSS del tema (`--scumm-verb-cyan`, `--scumm-verb-green`,
  `--scumm-verb-yellow`, `--scumm-wood`, `--scumm-wood-hi`).

Este directorio no se distribuye en el ZIP del estilo.
