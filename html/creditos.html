Créditos y descargas

Créditos

Recurso educativo sobre el ciclo del agua y tema visual SCUMM Adventure para eXeLearning.

Autoría

  • Área de Tecnología Educativa — diseño del tema, integración con eXeLearning, contenidos educativos y maquetación.

Tipografías

  • lucasarts-scumm-solid, lucasarts-scumm-menu, lucasarts-scumm-menu-solid y lucasarts-scumm-subtitle-roman (OTF) — fuente: scummbar.com. Uso no comercial, homenaje a LucasArts.
  • Fallback: Courier New y monoespaciadas del sistema — mantienen la estética retro si la OTF no se carga.

Sprites pixel art

  • Escena (sky, ocean, mountain, sun, cloud, rain, river), UI (cursor, frame, verb_button, inventory_slot), iconos iDevice e iconos de inventario — 29 sprites PNG 8/16-bit locales.

Integración con eXeLearning

  • Convenciones de config.xml, style.css y style.js inspiradas en los temas flux y universal del repositorio principal de eXeLearning.
  • Formato ODE 2.0 (content.xml + content.dtd) para el recurso empaquetado como .elpx.

Licencia

Todo el material (tema, sprites locales, contenidos) se distribuye bajo Creative Commons BY-SA 4.0, con los reconocimientos correspondientes a las tipografías LucasArts originales.

Descarga el código fuente

Este recurso es abierto. Puedes descargarlo, editarlo y redistribuirlo respetando la licencia CC BY-SA 4.0.

Paquetes disponibles

  • Recurso educativo (.elpx) — proyecto eXeLearning completo del ciclo del agua.
    ⬇ Descargar .elpx
  • Tema SCUMM Adventure (.zip) — para importar en Herramientas → Gestor de estilos → Importar estilo.
    ⬇ Descargar tema .zip
  • Código fuente completo — repo Git con todos los assets, el workflow CI y la versión descomprimida del .elpx.
    ⬇ Repositorio en GitHub

Abrir directamente en el navegador

Sin instalar nada, en el runtime estático de eXeLearning:

static.exelearning.dev/?url=…/el-ciclo-del-agua.elpx

Usar el tema sin el recurso

  1. Descarga el ZIP del tema desde la última release.
  2. En eXeLearning: Herramientas → Gestor de estilos → Importar estilo.
  3. En Propiedades → Estilo, selecciona SCUMM Adventure.
  4. Exporta tu proyecto como Sitio Web, SCORM, IMS o EPUB.

Si reutilizas el recurso o el tema, mantén el crédito al Área de Tecnología Educativa y enlaza a este repositorio. ¡Gracias!
